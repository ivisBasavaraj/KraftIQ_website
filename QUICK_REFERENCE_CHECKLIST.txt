COMMON MISTAKES TO AVOID
─────────────────────────────────────────────────────────────────────────────
✗ DON'T: Use multiple H1 tags (only one per page!)
✗ DON'T: Forget alt text on images
✗ DON'T: Use keyword stuffing (sounds unnatural)
✗ DON'T: Have broken links
✗ DON'T: Forget breadcrumb navigation
✗ DON'T: Use placeholder images (use real Unsplash/Pexels)
✗ DON'T: Skip schema markup
✗ DON'T: Make pages too long (2,000-3,000 words is good)
✗ DON'T: Forget mobile optimization
✗ DON'T: Skip internal linking
